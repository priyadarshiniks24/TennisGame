This is a simple browser based tennis game for two players, one being the computer and the other being the user.

How to start the game?
To start the game, open tennisGame.html using any browser.
Left player is the user and the right player is the computer.

How to play?
To control the player's paddle use mouse.

How to win?
Player scores when the opponent misses the ball.
Who ever scores 10 points first, wins the game.
